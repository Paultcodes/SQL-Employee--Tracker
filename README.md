# SQL-Employee-Tracker

## Description
This is a content managment system that lets you interact with the SQL database so that you can update, add, or change the data. It uses inquirer for the prompts and mysql2 to connect to the database.


## Getting Started

Clone the repository by copying the SSH and key and running "git clone" follwed by the coppied content in your terminal. To download the necessary dependencies run npm i.
